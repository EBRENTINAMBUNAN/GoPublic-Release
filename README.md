# GoPublic Release

Folder ini berisi artefak distribusi publik GoPublic.

## Struktur

- setiap versi memiliki folder sendiri, misalnya `v0.1.2/`
- di dalam folder versi tersedia binary, checksum, catatan rilis, installer, dan README versi

## Versi Terbaru

- `v0.1.3`

## Cara Pakai

1. Masuk ke folder versi yang ingin dipakai.
2. Baca `README.md` pada folder versi tersebut.
3. Jalankan installer publik atau gunakan binary secara manual.
