# GoPublic Release

Folder ini berisi artefak distribusi publik GoPublic.

## Struktur

- setiap versi memiliki folder sendiri, misalnya `v0.1.2/`
- file `LATEST` selalu menunjuk ke versi terbaru
- file `latest.json` menyimpan metadata release terbaru dalam format machine-readable

## Versi Terbaru

- `v0.1.6`

## Cara Pakai

1. Baca file `LATEST` atau `latest.json` untuk mengetahui versi aktif terbaru.
2. Masuk ke folder versi yang ingin dipakai.
3. Baca `README.md` pada folder versi tersebut.
4. Jalankan installer publik atau gunakan binary secara manual.
