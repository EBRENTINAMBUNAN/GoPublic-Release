# GoPublic v0.1.1

## Apa yang Baru

- Rilis binary Linux native pertama untuk distribusi publik GoPublic.
- Menyertakan artefak dasar release berupa binary `onefile`, checksum SHA-256, dan catatan rilis.
- Menjadikan GoPublic dapat dijalankan sebagai executable mandiri di Linux Debian/Ubuntu tanpa perlu menjalankan source code Python secara langsung.

## Artefak Rilis Ini

- binary: `gopublic-linux-amd64-v0.1.1`
- checksum: `gopublic-linux-amd64-v0.1.1.sha256`
- catatan rilis: `gopublic-linux-amd64-v0.1.1.txt`
- installer publik: `install-gopublic.sh`
- uninstaller publik: `uninstall-gopublic.sh`

## Catatan

- Folder ini hanya berisi artefak untuk rilis `v0.1.1`.
- Gunakan installer atau binary langsung dari folder ini jika Anda memang ingin memakai versi `v0.1.1`.
