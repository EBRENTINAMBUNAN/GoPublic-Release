Application : gopublic
Version     : 0.1.6
Platform    : linux
Arch        : amd64
Build Mode  : onefile
Binary      : gopublic-linux-amd64-v0.1.6

Ringkasan cepat:
- binary ini bersifat mandiri; user tidak perlu install source code Python
- sistem target yang direkomendasikan: Debian/Ubuntu amd64
- prasyarat wajib: sudo, akun Cloudflare, dan domain aktif di Cloudflare

Langkah awal yang disarankan:
1. sudo bash ./install-gopublic.sh
2. gopublic
3. gopublic menu
4. cd /path/to/project && gopublic init
5. gopublic add app
6. gopublic push

Catatan:
- registry binary disimpan di ~/.config/gopublic/projects.json
- dashboard utama tersedia via `gopublic dashboard`
- workflow manual lama tetap tersedia via `gopublic new`
